NPYScreen Documentation

Purpose

NPYScreen is a library for python. It is designed to make the writing of curses-based user-interfaces quick and painless.

Installation

NPYScreen is packaged using the standard python tools. You can type: python setup.py install to install. Version 2.5 or higher of python is required.

Basic use

Importing

Import the module with the line: import npyscreen

Basic Object Types

The library defines three basic kinds of object.

Widget Objects
This is the most numerous class: text fields, lists, date fields, and so on and so forth.
Form Objects
Forms are the containers for widgets. An application will have at least one form. Some forms also provide additional facilities, such as menus.
NPSApplication Objects
While you can program using npyscreen without using npsapplication objects, these provide an additional level of abstraction, and are the easiest way to use the library.
