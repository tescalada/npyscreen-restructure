Method Details [hide private]

handle_input(self, input)

source code 
Returns True if input has been dealt with, and no further action needs taking. First attempts to look up a method in self.input_handers (which is a dictionary), then runs the methods in self.complex_handlers (if any), which is an array of form (test_func, dispatch_func). If test_func(input) returns true, then dispatch_func(input) is called. Check to see if parent can handle. No further action taken after that point.

set_up_handlers(self)

source code 
This function should be called somewhere during object initialisation (which all library-defined widgets do). You might like to override this in your own definition, but in most cases the add_handers or add_complex_handlers methods are what you want.

add_handlers(self, handler_dictionary)

source code 
Update the dictionary of simple handlers. Pass in a dictionary with keyname (eg "^P" or curses.KEY_DOWN) as the key, and the function that key should call as the values
